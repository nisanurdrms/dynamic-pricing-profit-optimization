## İçerik

* Veri Temizleme
* Veri Entegrasyonu (Retail + CPI)
* Talep Esnekliği Analizi
* Feature Engineering
* Random Forest
* XGBoost
* Hyperparameter Optimization
* SHAP Analizi
* Kâr Optimizasyonu Simülasyonu
* Hurwicz Karar Kriteri
